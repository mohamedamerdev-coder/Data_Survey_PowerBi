# Data Survey Project

This project analyzes survey data in the Data field.

##  Contents
- **Average Salary**: Data Engineer, Data Analyst, Data Scientist, etc.  
- **Country Survey Takers**: Number of participants per country.  
- **Favorite Programming Language**: Most popular languages among participants.  

## Tools
- Power BI  
- Data files are in the **Data survey** folder

##  Goal
Quick insights on salaries, country participation, and favorite programming languages.
